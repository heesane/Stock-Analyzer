# Stock Analyzer – 기능 개요

## 대화형 CLI
- `You > … / Assistant > …` 형식의 대화 흐름과 스트리밍 출력.
- `--lang` 옵션으로 한국어/영어 인터페이스 전환.
- `--benchmark`, `--backtest` 전역 옵션을 통해 벤치마크/백테스트 기간 지정.

## 핵심 분석 기능
- **기술적 지표**: MACD, RSI, 이동평균 배열, 가격 채널, 거래량 스파이크, 변동성 등.
- **밸류·펀더멘털**: PER/PBR 기반 밸류에이션, EPS 성장률.
- **수급·심리**: 단기 머니플로우, 뉴스 감성 분석, 시장 모멘텀.
- **리스크 진단**: 30/60일 변동성, 180일 최대 낙폭, ATR 기반 손절선, 위험 등급 분류.
- **상대 성과**: 사용자 지정 벤치마크 대비 알파/수익률 비교.
- **간단 백테스트**: 지정 기간 buy & hold 수익률, 벤치마크 대비 알파, 승률 계산.

## 결과 보고
- “시장 스냅샷”, “신호 보드”, “보조지표 브리핑” 등 섹션을 카드 형태로 스트리밍 출력.
- 추가 카드: “섹터/벤치마크 비교”, “리스크 진단”, “백테스트 스냅샷”.
- 스코어카드는 각 지표 value(0~1)와 weight를 노출하며, 총점을 0~100, 등급을 강력 매수/매수/중립/매도로 구분.

## 내보내기 및 인터랙션
- JSON/CSV/DB(MySQL, PostgreSQL) 내보내기.
- 인터랙티브 모드에서는 방향키 메뉴로 저장 형식 선택 및 친절한 안내 메시지 제공.
